# 🚀 CareerPilot AI

CareerPilot AI is an AI-powered career recommendation platform that analyzes resumes and provides career guidance using Machine Learning and NLP.

## 🌟 Features

- 📄 Resume Analysis
- 🎯 Career Prediction
- 📊 ATS Score Analysis
- 🧩 Skill Gap Detection
- 📚 Personalized Learning Roadmap
- 💼 Internship Recommendations
- 📥 Downloadable PDF Reports
